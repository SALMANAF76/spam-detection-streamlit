# SMS Spam Detection (Streamlit App)
Aplikasi sederhana untuk mendeteksi apakah sebuah pesan SMS termasuk Spam atau Bukan Spam, menggunakan TF-IDF dan Naive Bayes. Aplikasi ini dibuat dengan Python, scikit-learn, dan Streamlit.

## Cara Menjalankan
- Klik tombol Code â†’ Download ZIP di GitHub.
- Ekstrak ZIP ke folder manapun.
- Buka folder tersebut menggunakan VS Code.
- Pastikan library yang diperlukan sudah terinstall: pip install -r requirements.txt
- Jalankan aplikasi dengan perintah: streamlit run spam.py

## Cara Kerja Singkat
- Teks SMS diproses menggunakan TF-IDF
- Model Multinomial Naive Bayes melakukan klasifikasi
- Hasil yang ditampilkan: Spam, Bukan Spam, Probabilitas spam
